Example URSYS App

About

The intent behind URSYS is develop reusable snippets of code that we used to build learning sciences research applications in a development environment that I've tuned to my liking over time. Because it's a monorepo, I can develop all the features at the same time across different packages.

This example app demonstrates an app based on URSYS. You can copy the example-app directory and rename it app or app-* to create a new working directory to play with.

The .gitignore file is set to ignore the app directories so you can initialize them as your own Git Submodule(s), each with its own build system and exports through package.json.

Repo Conventions

The repo uses the built-in npm workspaces feature with two defined workspaces:

  • Core is in the _ur directory, and are built as a library.
  • Addons are in _ur_addons, and are services built on top of Core.
The URSYS repo still in an early stage of development as features from old projects are being modernized for Typescript.
